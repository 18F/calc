CALC Style Guide Documentation

Welcome to the CALC Style Guide Documentation! Here you'll find guidelines on how to update the CALC Style Guide.

{% guide_section "Overview" %}

The style guide is a Django app in the {% pathname "styleguide" %} directory. The bulk of its textual content is in {% pathname "styleguide/templates/styleguide.html" %}, which is a standard Django template.

{% guide_section "Template tags" %}

The style guide comes with a number of template tags defined in {% pathname "styleguide/templatetags/styleguide.py" %}. In addition to being convenient, they're also designed to prevent documentation rot.
