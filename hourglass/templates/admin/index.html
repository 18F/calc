Manage pricing data

To review and exclude rows of user-submitted data: Select "Submitted price list rows," which contains all of the data that is not yet available in CALC. Note that this view shows data from multiple submitted price lists. Here you can "mute" rows that seem wrong or unusual to prevent them from showing up in CALC's data explorer.

To edit new, not-yet-published, user-submitted data: Select "Unreviewed price lists" to see a list of all new, unpublished price lists. Click on the number of the list you wish to edit to be taken to a detail view.

To approve and publish user-submitted data: Once you have muted any undesirable rows and edited any vendor or pricing data that require it, return to "Unreviewed price lists" to approve or reject pricing data.

To reject user-submitted data: From the "Unreviewed price lists" view, select the price list you want to reject and choose "reject." (You can approve a rejected price list later from the "Rejected price lists" view if you reject a list by mistake!)

To remove user-submitted data: Select "Approved price lists" and choose the price list you want to remove from CALC. This is referred to as "retiring" a price list. The price list will remain available in the admin view, but will no longer be displayed to users in the data explorer.

To edit data that has already been published: First, remove the data from public use by retiring it as explained above. View "Retired price lists" and click on the contract number of the desired list. Make any necessary edits, and then return to the "Retired price lists" view to approve and re-publish the pricing data.

To help troubleshoot data submission attempts: Select "Attempted price list submissions" for a list of submissions that never made it all the way to completion. From here, you can "replay" submission attempts to try to see what glitches the user submitting data might have hit.
